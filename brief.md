## 1. ЦЕЛЬ И АУДИТОРИЯ

### Цель сайта
Конвертировать незнакомого посетителя в заявку на сотрудничество (звонок / сообщение) за одну сессию. Вторичная цель — сформировать доверие к Александру как к живому эксперту, а не безликому фрилансеру.

### Целевая аудитория

| Сегмент | Боль | Что ищет |
|---------|------|----------|
| **Основатель стартапа** (28–42 лет) | Нет ресурсов на полноценную команду | Быстрый запуск MVP, один ответственный контакт |
| **Владелец малого бизнеса** (35–50 лет) | Рутина съедает время, прошлые подрядчики подводили | Автоматизация процессов, надёжность, понятная цена |
| **Маркетолог / продакт в компании** | Нужен технический партнёр под конкретную задачу | Скорость, коммуникация, выполнение в срок |
| **Предприниматель, интересующийся AI** | Слышал про ИИ-агентов, не понимает как применить | Объяснение простым языком + готовое решение |

### Главный инсайт из исследования
80%+ трафика с мобильных. Решение «остаться или уйти» принимается за 5 секунд. Клиент покупает у человека, а не у специальности — фото и личность критичны.

## 5. ОБЯЗАТЕЛЬНЫЕ ЭЛЕМЕНТЫ

### Анимации

| Элемент | Анимация | Реализация |
|---------|----------|------------|
| Секции при скролле | Fade-up (opacity 0→1, translateY 40px→0) | CSS `animation-timeline: view()` |
| Заголовок H1 | Посимвольное появление | CSS stagger или SplitType + GSAP |
| Карточки услуг | Staggered reveal (задержка 0.1s между) | CSS `animation-delay` |
| Счётчики в trust bar | Count-up при попадании в viewport | Intersection Observer + JS |
| Hover на карточках | Lift + glow | CSS transition |
| Фон hero | Медленное движение aurora | CSS `@keyframes` background-position |
| Появление sticky-header | Slide-down при скролле | JS scroll listener |

**Уважение к `prefers-reduced-motion`:**
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

### Интерактивные элементы

- **Custom cursor** (desktop): точка 6px + окружность 36px, расширяется на hover над кнопками
- **Magnetic buttons** (desktop): главные CTA-кнопки притягиваются к курсору в радиусе 80px
- **Smooth scroll**: Lenis v2 (плавная инерция скролла)
- **Фильтр в портфолио**: переключение тегов с анимацией (CSS `display: none` → `opacity`)

### Формы

- Форма контакта: 3 поля (имя / контакт / задача), `required` валидация
- Submit: loading state (spinner в кнопке) → success message без перезагрузки
- Honeypot-поле против спама (скрытое поле `name="_gotcha"`)
- Бэкенд: Web3Forms (бесплатно, без сервера) или Formspree
